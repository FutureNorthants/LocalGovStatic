To add links to the footer menu, create a `footer_links.yml` file in the `_data` directory and add links using the following format:

```yaml
items:
  - name: Templates
    link: /templates/
  - name: Components
    link: /components/
  - name: News
    link: /news/
  - name: Cookie policy
    link: /cookie-policy/
```
